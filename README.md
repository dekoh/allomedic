allomedic
=========

mon tfe

Le défi est que mon tfe soit entièrement fonctionel. 

Pour que tout fonctionne bien il faut bien regarder a modifier les différentes choses suivantes:
1. Dans le fichier config.inc.php, bien préciser l'url de base ainsi que les données de connexion à la base de donnée. 
2. Dans le fichier .htaccess, préciser l'url de base, peut etre /tfe/ comme dans ce cas si. Si c'est a la base de l'url il vaut mieux le préciser(exemple:allomedic.com).

Bonne lecture!
